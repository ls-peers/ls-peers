# ls-peers

## Running locally
### With Bundle
1) Clone repo
2) `$ cd` into repo
3) Run `$ bundle install`
4) Run `$ bundle exec ruby peers.rb`

### With Heroku
1) Clone repo
2) `$ cd` into repo
3) Run `$ bundle install`
3) Run `$ heroku local`

## Deploying to production
### With Github
1) Merge branch with main (Heroku auto deploys changes to main)

### With Heroku
1) Run `$ git push heroku [branch-name]`
